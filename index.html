EventPort Javascript implementation.

This project is a basic js implementation of an EventPort API loosely based on https://github.com/WICG/input-for-workers/pull/8
This is only a testing project meant to better see how such an API should be shaped.
Do not use in production, as it hasn't been tested, and probably won't be.

Currently doesn't work between cross-origin documents.

Test pages:

  • Control an OffscreenCanvas from a deep Worker using the mouse.
  • Control an canvas in an iframe or a popup window using the mouse.
  • Control an AudioWorkletProcessor using the mouse.
